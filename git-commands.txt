чтобы понять статус репозитория пишем (создать/редактировать файлы)
проверить наличие новых/редактированных файлов:
git status

подготовить файлы, которые попадут в коммит (подготовить набор файлов для коммита):
git add <название файла>
git add --all

чтобы закоммитить:
git commit -m "описание коммита"

чтобы посмотреть историю коммитов, пишем:
git log

чтобы перейти к конкретному коммиту
git  checkout <hash>

Чтобы связать репозиторий с локальным один раз 
git remote add origin <адрес репозитория>

чтобы глянуть список удаленных репозиториев
git remote --v

чтобы залить код на удаленный репозиторий:
git push origin master

чтобы подтянуть код с удаленного репозитория:
git pull origin master
