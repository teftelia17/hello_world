чтобы понять статус репозитория пишем (создать/редактировать файлы)
проверить наличие новых/редактированных файлов
git status

подготовить файлы, которые попадут в коммит (подготовить набор файлов для коммита)
git add <название файла>
git add --all

чтобы отправить файлы в git (создать коммит)
git commit -m "описание коммита"
